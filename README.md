# Raspberry Pi Pico MicroPython Examples

This repository contains a collection of **MicroPython code examples for the Raspberry Pi Pico (and Pico W)**.  
The examples are organized in a **step-by-step learning path**, starting from basic GPIO control and progressing to communication protocols, motor control, and networking.

Each folder contains its own detailed `README.md` explaining the code, hardware setup, and pin connections.

---

# Learning Roadmap (Basic → Advanced)

### 1. LED (Digital Output)
Basic GPIO output to blink an LED and understand how to control pins.

### 2. Button (Digital Input)
Read push button states and trigger actions like turning an LED ON/OFF.

### 3. ADC (Analog to Digital Converter)
Read analog voltages from components like potentiometers and convert them into digital values.

### 4. PWM (Pulse Width Modulation)
Generate PWM signals to control LED brightness and other devices.

### 5. Servo Motor
Control hobby servo motors using PWM signals and move them to specific angles.

---

# Communication Protocols

### 6. UART
Basic serial communication for sending and receiving data.

### 7. I2C
Interfacing with I²C devices such as **SSD1306 OLED displays**.

### 8. SPI
SPI communication examples including transmit, receive, and loopback testing.

---

# Networking (Advanced)

### 9. WiFi Connect (Pico W Only)
Connect the **Raspberry Pi Pico W** to a Wi-Fi network and host a simple HTTP web server to control an LED.
